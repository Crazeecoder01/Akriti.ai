## üåü Project Overview

### Problem Statement

Web development has a steep learning curve. Designers sketch beautiful interfaces, but bringing them to life requires extensive HTML, CSS, and JavaScript knowledge. This gap between design and implementation slows down prototyping and makes web development inaccessible to non-coders.

### Our Solution

Akriti.ai democratizes web development by using AI to instantly convert hand-drawn sketches into production-ready websites. Our intelligent system:

- **Understands Context**: Reads text in your sketches to understand the website's purpose, industry, and brand
- **Fetches Real Assets**: Automatically retrieves actual company logos and relevant images
- **Generates Smart Code**: Creates responsive, accessible, and modern HTML with Tailwind CSS
- **Enables Refinement**: Allows natural language instructions to modify the generated website

### Key Innovation

Unlike traditional website builders that use pre-made templates, Akriti.ai uses **Google Gemini Vision AI** to truly understand your sketch and generate unique, contextually appropriate websites that match your exact vision and brand identity.

##  Getting Started

### Prerequisites
- Node.js 20+ installed
- npm or yarn package manager
- Google AI API key (for Gemini AI)
- Tldraw license key (for production deployment)

### Installation

1. Clone the repository:
```bash
git clone https://github.com/Crazeecoder01/Akriti.ai.git
cd akriti.ai
```

2. Install dependencies:
```bash
npm install
```

3. Set up environment variables:

Create a `.env.local` file in the root directory and add the following:

```bash
# Google AI API Key (required)
GOOGLE_API_KEY=your_google_api_key_here

# Tldraw License Key for production deployment (required)
NEXT_PUBLIC_TL_DRAW_LICENSE_KEY=your_tldraw_license_key_here
```

> **Note**: You can use the `.env.example` file as a template.

4. Run the development server:
```bash
npm run dev
```

5. Open [http://localhost:3000](http://localhost:3000) in your browser

### Environment Variables

| Variable | Description | Required |
|----------|-------------|----------|
| `GOOGLE_API_KEY` | Your Google AI API key for Gemini AI | Yes |
| `NEXT_PUBLIC_TL_DRAW_LICENSE_KEY` | Tldraw license key for production | Yes (for production) |

**How to get API keys:**
- **Google AI API Key**: Get from [Google AI Studio](https://makersuite.google.com/app/apikey)
- **Tldraw License Key**: Get from [Tldraw](https://tldraw.dev) for production deployments
